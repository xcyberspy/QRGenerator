<h2>🧐 Features</h2>

Here're some of the project's best features:

*   Generate high-quality QR codes with error correction
*   Detect URLs automatically or use plain text mode
*   Choose custom colors for your QR codes
*   Add logos to QR codes (max 5000x5000 px)
*   Live preview of generated QR codes
*   Modern dark UI with smooth performance

<h2>🛠️ Installation Steps:</h2>

<p>1. Clone the repository:</p>

```
git clone https://github.com/xcyberspy/QRGenerator.git
```

<p>2. Navigate to the Directory</p>

```
cd QRGenerator
```

<p>3. Install Required Packages</p>

```
pip install customtkinter qrcode Pillow pyperclip validators
```

<p>4. Run the application</p>

```
python QR Generator.py
```
